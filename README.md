# 🎡 Student Spin Wheel

A simple web application that randomly selects a student using a spinning wheel. This project is useful for classroom activities, quizzes, and random student selection.

## ✨ Features

* Random student selection
* Interactive spin wheel
* Simple and easy-to-use interface

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🚀 How to Run

1. Clone the repository.
2. Open the project in **Visual Studio Code**.
3. Open `index.html` in your browser.
